Short Description: Generation temperature - response creativity

Type: float

Default: 1.0

Range: 0.0 to 2.0

Details: This parameter affects the diversity of model responses. Low values lead to more predictable and typical responses, while high values promote more diverse and less common responses. At a value of 0, the model always gives the same response for a given input.

Short Description: Nucleus sampling - alternative to temperature

Type: float

Default: 1.0

Range: 0.0 to 1.0
